# AI Product Description Generator PRO - Chrome Extension

Generera SEO-optimerade produkttexter direkt i webbläsaren!

## Installation

### Utvecklingsläge
1. Öppna Chrome och gå till `chrome://extensions/`
2. Aktivera "Utvecklarläge" (Developer mode) i övre högra hörnet
3. Klicka på "Läs in okomprimerat tillägg" (Load unpacked)
4. Välj mappen `chrome-extension`

### Från Chrome Web Store
*(Kommer snart)*

## Funktioner

### 🚀 Popup Generator
- Ange produktnamn och nyckelord
- Välj språk (Svenska, Engelska, Norska, Danska, Finska, Tyska)
- Få SEO-optimerad text på sekunder
- Kopiera till urklipp med ett klick

### 🛒 E-handelsintegration
- Automatisk produktigenkänning på Shopify & WooCommerce
- Infogad "Generera AI-text"-knapp på produktsidor
- Extraherar automatiskt produktinformation från sidan

### ⚡ Snabbgenerering
- Högerklicka på markerad text
- Välj "Generera produkttext för markerad text"
- Resultat visas direkt på sidan

## Användning

1. **Logga in** med ditt AI Product Description Generator PRO-konto
2. **Navigera** till en produktsida eller öppna tillägget
3. **Generera** text genom att ange produktnamn eller använda sidan
4. **Kopiera** den genererade texten till din butik
